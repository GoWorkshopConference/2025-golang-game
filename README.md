# golang-game
Go の Ebitengine で作るミニゲーム

## 開発手順

1. このリポジトリをクローンしてきてください
2. 素材となる画像をダウンロードしてきて、`internal/assets` 以下に配置してください
   - 展示したゲームではいらすとやさんから以下の画像を使用しました。
   - [エビフライのイラスト（1本）](https://www.irasutoya.com/2019/09/1.html)
   - [お子様ランチのイラスト](https://www.irasutoya.com/2014/02/blog-post_6966.html)
